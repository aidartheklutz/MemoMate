# MemoMate

MemoMate is a mobile app designed for people living with memory issues such as dementia or Alzheimer's disease, set up and managed by their caretaker. We built it as team Brynza during Nexel Hackathon in under 9 hours to give vulnerable individuals a simple, dignified tool for navigating daily life.

Instead of juggling multiple apps and reminders, a caretaker can set up MemoMate once and leave their loved one with four core tools in one place: a to-do list for daily tasks, a memory-training game to keep the mind active, an AI chatbot designed specifically for users with memory difficulties, and a phonebook for reaching the people who matter most.

You can download an Android build of the prototype here: https://expo.dev/accounts/aidartheklutz/projects/memomate/builds/c96e95c1-2b1f-4b57-b074-801fb5582071

Please keep in mind that all of this was written under intense time pressure.

## Tech Stack

- **React Native** + **Expo** (cross-platform mobile development)
- **React Navigation** + **Native Stack** (screen navigation)
- **AsyncStorage** (local persistent storage)
- **Expo Notifications** (reminders and alerts)
- **Expo Image Picker** (photo support for the phonebook)
- **DateTimePicker** (scheduling for to-do items)
- **Expo Vector Icons** (UI iconography)

## Development

```
npm install
npx expo start
```

To run on a device or emulator:

```
npx expo run:android
npx expo run:ios
```

Some parts of the project were created with the assistance of AI.
